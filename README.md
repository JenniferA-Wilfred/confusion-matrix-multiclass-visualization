# 📊 Visualizing Confusion Matrix for Multi-Class Classification

## 📌 Overview
This project demonstrates multi-class classification using a Deep Learning model (Neural Network) and evaluates its performance using a confusion matrix. The model is trained on the Iris dataset and visualized using heatmaps and performance graphs.

## 🎯 Objectives
- Perform multi-class classification using a neural network  
- Evaluate model performance using confusion matrix  
- Visualize classification results using heatmaps  
- Analyze accuracy, precision, recall, and F1 score  

## 🧠 Model Architecture
- Input Layer: 4 features (Iris dataset)
- Hidden Layer 1: 16 neurons (ReLU activation)
- Hidden Layer 2: 8 neurons (ReLU activation)
- Output Layer: 3 neurons (Softmax activation)

## 🛠️ Technologies Used
- Python  
- TensorFlow / Keras  
- Scikit-learn  
- NumPy  
- Matplotlib  
- Seaborn
